# Pflanzenflaeche

Dieses Repository enthaelt die App `PlantAreaAnalyzer` im Ordner
[`PlantAreaAnalyzer/`](PlantAreaAnalyzer/).

Wenn du die App benutzen willst, brauchst du nicht das ganze Repo zu verstehen.
Der einfachste Einstieg ist:

1. [INSTALLATION_WINDOWS.md](PlantAreaAnalyzer/INSTALLATION_WINDOWS.md) lesen
2. Python installieren
3. das Projekt als ZIP herunterladen oder mit Git klonen
4. die App starten

Git ist nicht noetig, wenn du nur die App nutzen willst. Du brauchst Git nur
dann, wenn du spaeter Updates bequem mit `git pull` holen moechtest.

Die Bedienung der App steht hier:
[`PlantAreaAnalyzer/ANLEITUNG.md`](PlantAreaAnalyzer/ANLEITUNG.md)

Die Windows-Installation steht hier:
[`PlantAreaAnalyzer/INSTALLATION_WINDOWS.md`](PlantAreaAnalyzer/INSTALLATION_WINDOWS.md)

## Was ist wo?

- `PlantAreaAnalyzer/` - die eigentliche App
- `PlantAreaAnalyzer/analysis/` - Bildanalyse
- `PlantAreaAnalyzer/app/` - Benutzeroberflaeche
- `PlantAreaAnalyzer/data/` - Beispiel- und Referenzdaten
- `PlantAreaAnalyzer/exports/` - CSV-Exports
- `PlantAreaAnalyzer/tests/` - Tests
